✓ Configuration templates

🚀 QUICK START (3 STEPS):

Step 1: Run SQL scripts
  → sql/01_create_tables.sql
  → sql/02_create_stored_procedure.sql  
  → sql/03_create_views.sql

Step 2: Deploy application
  → Copy cfm/, css/, js/ to web server
  → Create uploads/ and exports/ directories
  → Set folder permissions

Step 3: Configure ColdFusion
  → Create datasource in CF Admin
  → Update datasource name in .cfm files
  → Test: http://your-server/timekeeping/

📊 KEY FEATURES:

• Upload Excel timekeeping reports

• Directorate/division enrichment
